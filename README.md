# ⚔️ Tyrants of the Underdark - Digital Edition

Cyfrowa wersja gry planszowej Tyrants of the Underdark.

## Jak grać

### Lokalnie (hot-seat)
1. Otwórz `index.html` w Chrome
2. Wybierz "👥 Lokalny"
3. Wczytaj karty (🃏) i mapę
4. Wybierz talie, sektory, graczy → Rozpocznij

### Online (multiplayer)
1. Otwórz stronę w Chrome
2. **Host**: "🏠 Stwórz pokój" → wyślij kod koledze
3. **Gracz**: "🔗 Dołącz" → wklej kod → Połącz
4. Obaj wczytajcie ten sam plik kart
5. Host wybiera talie/sektory → Rozpocznij

## Skróty klawiszowe
- **Spacja** — Zakończ turę
- **Enter** — Jestem gotowy
- **Z** — Zagraj wszystkie karty
- **Ctrl+Z** — Cofnij (tylko offline)
- **M** — Wycisz dźwięk
- **Tab** — Pokaż skróty
- **~** — Panel debug
- **1-9** — Wybierz opcję
- **Prawy klik** — Zagraj/Kup kartę

## Struktura plików
```
index.html          — gra
sounds/             — dźwięki
cards/              — obrazki kart (do dodania)
card-editor.html    — edytor kart
map-editor.html     — edytor mapy
```
